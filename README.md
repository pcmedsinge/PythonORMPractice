# Python ORM Practice Workspace

Hands-on PostgreSQL practice workspace for Python ORM frameworks:
- SQLAlchemy
- Django ORM
- SQLModel

The project is organized into phase-based learning tracks. Every phase includes:
- concept documentation
- runnable sample code
- lab exercises
- solved reference implementations

Database backend: PostgreSQL in Docker on host port 55432.

## Tracks
- sqlalchemy-phases/
- django-orm-phases/
- sqlmodel-phases/

Each phase contains:
- docs/FEATURES.md
- sample_code/
- labs/README.md
- labs/solutions/

## What You Can Practice
- CRUD with typed models and sessions
- one-to-many and many-to-many relationships
- joins, filtering, and reporting queries
- transactions and rollback behavior
- migrations and mini-project patterns
- SQLModel + FastAPI integration basics
