# Labs: Phase 02 Relationships and Joins

Difficulty: intermediate

## Exercise 1
Create models Author and Book with one-to-many relation. Insert 2 authors and at least 4 books.

Expected check:
- print each author with list of book titles
- filter books by author name using a join

## Exercise 2
Add many-to-many relation between Student and Club.

Expected check:
- list all students in a specific club
- list all clubs for a specific student
